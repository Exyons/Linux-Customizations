# Linux Customizations

Welcome to **Linux Customizations**! This repository contains a collection of scripts, configuration files, and tools to streamline the setup and customization of Linux environments, specifically targeted toward Arch Linux (CachyOS) and Fedora.

## Features

- **Fedora Customizations**: Scripts for deploying a beastly Hyprland environment, configuring Waybar/Hyprpanel, and customizing GRUB themes.
- **Arch/CachyOS Customizations**: Scripts to quickly configure and setup a customized CachyOS environment.
- **Utility Scripts**: Backup scripts and tools to maintain your Linux installations.

## Folder Structure

- `CachyOS/`: Scripts handling the setup and customization of CachyOS.
- `fedora_customization/`: Deployment scripts for Fedora, including Hyprland and aesthetic setups.
- `backup_script.sh`: A tool for backing up critical system data.
